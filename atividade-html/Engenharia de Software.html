Engenharia de Software - 2º Ano do Ensino Médio

Introdução à Engenharia de Software

Nesta seção, vamos explorar os conceitos básicos da engenharia de software, incluindo:

  1. Definição de engenharia de software e seu papel na tecnologia
  2. Principais atividades do ciclo de vida do desenvolvimento de software: especificação, projeto, implementação, testes, manutenção
  3. Modelos de processo de desenvolvimento de software: cascata, espiral, ágil, incremental, entre outros
  4. Importância da engenharia de requisitos na especificação de sistemas de software
  5. Padrões de qualidade de software: ISO 9126, CMMI, MPS.BR
  6. Tendências atuais em engenharia de software: DevOps, Continuous Integration/Continuous Deployment (CI/CD), Inteligência Artificial aplicada ao desenvolvimento de software

Processos e Métodos Ágeis

Aqui, vamos estudar os processos e métodos ágeis na engenharia de software, incluindo:

  1. Princípios do Manifesto Ágil e sua aplicação prática
  2. Scrum: papéis (Product Owner, Scrum Master, Time de Desenvolvimento), eventos (Sprint, Reunião de Planejamento da Sprint, Revisão da Sprint, Retrospectiva da Sprint), artefatos (Product Backlog, Sprint Backlog, Incremento), práticas do framework Scrum
  3. Extreme Programming (XP): valores (Comunicação, Simplicidade, Feedback, Coragem, Respeito), práticas (Programação em Pares, TDD - Test-Driven Development, Integração Contínua, Refatoração), princípios do XP
  4. Kanban: princípios (Visualizar o Trabalho, Limitar o Trabalho em Progresso, Gerenciar o Fluxo, Tornar as Políticas de Processo Explícitas, Feedback Rápido), práticas do método Kanban
  5. Comparação entre métodos tradicionais (cascata, incremental) e métodos ágeis
  6. Adoção de práticas ágeis em diferentes contextos organizacionais

Desenvolvimento de Software

Nesta seção, vamos explorar técnicas e ferramentas de desenvolvimento de software, incluindo:

  1. Programação orientada a objetos: conceitos básicos (classe, objeto, herança, polimorfismo, encapsulamento), princípios SOLID
  2. Desenvolvimento de interfaces de usuário: princípios de design de interfaces, ferramentas de prototipação
  3. Gestão de configuração de software: controle de versão (Git, SVN), gerenciamento de mudanças
  4. Testes de software: tipos de testes (unitários, integração, sistema, aceitação), estratégias de teste, automação de testes (JUnit, Selenium)
  5. Desenvolvimento de software para web: HTML, CSS, JavaScript, frameworks front-end (React, Angular, Vue.js), frameworks back-end (Node.js, Spring Boot, Django)
  6. Desenvolvimento de aplicativos móveis: plataformas (iOS, Android), ferramentas de desenvolvimento (Android Studio, Xcode), frameworks (React Native, Flutter)
  7. Desenvolvimento de software orientado a serviços: arquitetura de microsserviços, APIs RESTful, gRPC

Gestão de Projetos de Software

Aqui, vamos discutir sobre a gestão de projetos de software, incluindo:

  1. Ciclo de vida de um projeto de software: iniciação, planejamento, execução, monitoramento e controle, encerramento
  2. Metodologias de gestão de projetos: PERT/CPM, método do caminho crítico, diagrama de Gantt, SCRUM, Kanban
  3. Ferramentas de gestão de projetos: Microsoft Project, Trello, Jira, Asana, Monday.com
  4. Desafios e boas práticas na gestão de projetos de software
  5. Gestão de equipes de desenvolvimento: liderança, motivação, comunicação, resolução de conflitos
  6. Gestão de requisitos: elicitação, análise, documentação, rastreabilidade
